# 3D Challenge Level Overrides
## Jul 7 2010

Replace 00000000.app address 0x1385CC with 0X 00 A0 E3, where X is:
0 -> ninja
1 -> mii runner
2 -> choco
3 -> strawberry
4 -> dance
5 -> golf
6 -> minecart

## Jul 12 2010

Replace 00000000.app address 0x2505C with 0X 00 A0 E3, where X is:
0 -> ninja
1 -> mii runner
2 -> chocolate
3 -> strawberry
4 -> dance
5 -> golf
6 -> minecart
7 -> hurdle
8 -> basket
9 -> sideways basket
A -> frog
